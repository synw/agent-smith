## Usage

Get the default model: <kbd>minicpm-v:8b-2.6-q8_0</kbd> (Ollama)

To run an inference query using images use the <kbd>vision</kbd> job:

```bash
lm vision imgs/img1.jpeg imgs/img2.jpeg p="Compare the images"
```
